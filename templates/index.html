PaccanaroLab's Disease similarity browser

Great advances have been made over the past decades and we have more information than ever before. This vast amount of proteomic data has helped clarify the relationship between a disease phenotype and its causes, to a point where it is now clear that a disease is a wider perturbation in the underlying biological networks.

There is relatively little molecular information about the hereditable diseases described in OMIM, with very few causal genes identified for most diseases and none for more than 45% of them. On the other hand there is an abundance of phenotype descriptions that, although not well suited for machine processing, provide comprehensive information on the various diseases.

We designed a method to extract the information contained in OMIM and, using the structure provided by MeSH, obtain a single number to characterise the similarity of any two genetic diseases. Our method is able to integrate the different domains of knowledge contained in MeSH, and we can show that our similarity measure reflects the common molecular machinery associated to similar diseases.

We have developed this browser where users can query pairs of diseases or explore a diseases "neighbourhood". We have made all data and code available for Download.

If you found our data useful, please cite:
An ontology–based measure of similarity between hereditable disease
 Horacio Caniza, Alfonso E. Romero and Alberto Paccanaro

Search!

Type the name or MIM number of the diseases you want to compare and press 'Search' to obtain the results. If you want to test the browser with the provided example, just click the 'Search' button.



Explore!

We have developed a way to explore a diseases' "neighbourhood". Select the disease you want by typing its name or MIM number and click 'Explore'. You can test the explorer with the provided example by just clicking 'Explore'.


Download

  • Disease similarity file in matrix form
  • Disease similarity file in triplet form
  • OMIM to MeSH mapping
The matrix file presents all results in a square real marix. Every row and every column corresponds to a disease in OMIM. The corresponding cell between a row and a column contains a real value which represents the similarity between the respective diseases.
The list file shows the similarity between two elements by pairing them up one by one. Every line consists of three columns. In case you selected a genewise calculation:
  • Column 1 are MIM numbers
  • Column 2 are MIM numbers
  • Column 3 contains the value representing the similarity between the elements in column 1 and column 2
The OMIM to MeSH file is a tab separated file in which the first column is the OMIM disease identifier and the remaining columns are the MeSH terms unique identifiers assigned to it.
